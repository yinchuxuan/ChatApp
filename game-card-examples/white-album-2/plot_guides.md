# 剧情引导

## 三人初识

### 剧情节点
三人初识

### 时间范围
2007.10.20: 15:00 - 2007.10.25: 17:00

### 人物状态
冬马和纱和春希虽然是同桌，在之前有过一些交集，但此时冬马对春希的态度仍然十分冷淡，像在有意回避什么
小木曾雪菜只是听说过春希，没有和春希有过什么实质性的接触，彼此之间相对陌生

### 剧情目标
让北原春希通过《WHITE ALBUM》先后与冬马和纱、小木曾雪菜产生联系。此窗口没有平行主线，必须固定推进到春希在天台遇到雪菜，并让春希意识到三人的音乐可以组成学园祭演出的核心。

### 按时间顺序必须出现的剧情节点
1. 春希所在的轻音乐同好会濒临解散，只剩下春希和武也两个人；春希想在学园祭上登台表演，但是同好会的现状并不支持他们报名拿出演出节目
2. 春希在第三音乐室独自练习吉他, 隔壁传来弹奏的《WHITE ALBUM》, 春希与隔壁钢琴完成一次隔墙合奏《WHITE ALBUM》; 春希在练习吉他时一直有隔壁钢琴相伴，但不知道隔壁是谁在弹钢琴
3. 春希作为学园祭代理执行委员去邀请小木曾雪菜参加学园祭的“峰城大附属小姐”的评选，第一次和雪菜接触
4. 春希试图接近冬马，但冬马一直对他爱搭不理
5. 到了2007.10.25: 10:00, 今天是学园祭报名节目的截止日期，春希自知无法拿出演出节目，作为代理执行委员在学园祭的演出名单上将轻音乐同好会这一栏划掉
6. 2007.10.25: 17:00，春希又一次来到第三音乐室练习吉他，心里想着做最后一次告别；在弹奏《White Album》时，隔壁的钢琴又加入了合奏；就在春希在心里感谢隔壁同学一直以来的陪伴时，天台上响起了第三个声音——宛如天籁般的《White Album》歌唱声；春希心里一阵悸动，对隔壁同学大喊"拜托了隔壁同学，不要停下"，一边狂奔上天台；在天台看见在夕阳下的小木曾雪菜正在高声歌唱

### 互动要点
1. 2007.10.26之前春希无论如何都无法获知隔壁弹钢琴的人是冬马
2. 2007.10.26之前冬马和雪菜都不会同意加入学园祭的演出
3. 雪菜是否同意参加“峰城大附属小姐“评选视互动情况而定
4. 2007.10.25: 17:00必须推进到三人《WHITE ALBUM》合奏、天台遇见雪菜这个核心节点
